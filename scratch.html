Introduction and Requirments

To begin your journey in scratch: there are some stuff you need. first of all, i advise you getting headphones or earplgs as when you use the sound section a lot of noise will be made. You also need the scratch software, whether online or the scratch
3 app: both are fine: I would advise the online version beacuse any updates will immediately be put there. You might be tempted to goi to the scratch website and hget all the source code from there, but you'll soon realise that next time you won't be able to code that easily beacuse you probably forgot most of the code.
